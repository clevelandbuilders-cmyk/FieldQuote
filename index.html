How to Get a Fresh Cookie
  • 1Open PellaPro on your Mac in Chrome and log in
  • 2Press Command + Option + I → Network tab → Fetch/XHR filter
  • 3Configure any window to trigger an API call
  • 4Click any api.pella.com request → Headers → Request Headers
  • 5Copy the entire cookie: value
  • 6Go to dash.cloudflare.com → Workers & Pages → fieldquote-proxy
  • 7Click Settings → Variables and Secrets
  • 8Edit PELLA_COOKIE → paste new value → Deploy
